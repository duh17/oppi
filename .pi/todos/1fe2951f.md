## Deliverables

1. **Workspaces as top-level nav** — replace session-first list with
   workspace list showing running/stopped state and session counts.

2. **Workspace detail** — shows session states (Running, Stopped, Resuming),
   new session / resume / stop actions via workspace-scoped APIs.

3. **Keep existing reliability** — reconnect, state restoration, and
   timeline rendering must not regress.

- No regression in timeline rendering (tool/thinking/text)
