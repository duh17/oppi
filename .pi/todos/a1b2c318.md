## Summary (NNW §3.2)

Extract a shared `PiRemoteCore` Swift package that both iOS and macOS apps import:

- Models (Session, Permission, ServerMessage, etc.)
- Protocol decode (ServerMessage decoder, Codable implementations)
- Reducer/coalescer (TimelineReducer, DeltaCoalescer)
- API/WS clients (APIClient, WebSocketClient, ServerConnection)

## Why

NetNewsWire uses shared core packages across iOS and Mac. This prevents drift, reduces duplication, and makes the macOS app mostly a thin UI shell.

## Approach

- Create `PiRemoteCore` as a local Swift package
- Move all `Core/` directory contents into it
- Keep platform-specific UI shells thin
- Shared fixtures + decode tests as package tests
