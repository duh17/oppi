## Context

After the agent builds a skill in a session (creates SKILL.md + scripts
in /work/<name>/), the user needs a way to save it to persistent storage.

Depends on: TODO-eabe2bf3 (server skill CRUD), TODO-15956340 (file browser)
Tracker: TODO-992ad1a6

## What to do

### A. Skill detection in file browser

When viewing session workspace files (Output tab), if a subdirectory
contains a SKILL.md file, show a "Save as Skill" action:

- In FileListView, detect directories that contain SKILL.md
- Show a banner or highlighted row: "📦 data-viz — Save as Skill"
- Alternatively, add a toolbar button "Save as Skill" that lists
  detected skill directories

### B. Save confirmation sheet

When user taps "Save as Skill":

1. Show sheet with:
   - Skill name (from directory name, editable)
   - Description (extracted from SKILL.md frontmatter, read-only preview)
   - File list + total size
   - [Save] [Cancel] buttons

2. On Save:
   - Call `POST /me/skills { name, sessionId }`
   - Show success toast
   - Navigate to Skills tab or skill detail

### C. SkillService.saveSkill

Add to SkillService:
```swift
func saveSkill(name: String, sessionId: String) async throws -> Skill
```
