## Context

Protocol plumbing is complete — 28 ClientMessage variants, ServerMessage routing, AgentEvent pipeline, TimelineReducer handling. All data flows end-to-end but there are no UI controls to trigger the new commands.

## Completed

### 1. Session Toolbar (DONE)
New `SessionToolbar.swift` — compact interactive strip below nav bar:
- **Model picker**: tap current model name → opens ModelPickerSheet
- **Thinking level**: tap to cycle (off → low → medium → high → off), haptic feedback
- **Context usage**: mini progress bar + percentage/window (color-coded: green/orange/red)
- **Cost display**: running session cost
- **Overflow menu**: Compact Context, Rename Session, New Session

### 2. Model Picker Sheet (DONE)
New `ModelPickerSheet.swift`:
- Fetches models from REST API (`GET /models`)
- Groups by provider (Anthropic, OpenAI, Google, etc.)
- Search/filter across model name, ID, and provider
- Current model highlighted with "current" badge + checkmark
- Shows context window size per model
- Tap to select → sends `set_model` via WebSocket

### 3. Session Management (PARTIAL)
- **New session**: available from toolbar overflow menu, sends `new_session` RPC
- **Fork**: context menu on user/assistant messages in SessionOutlineView → sends `fork(entryId:)`
- **Session name editing**: rename alert from toolbar overflow menu → sends `set_session_name`

### 4. Bash Input (DONE)
Modified `ChatInputBar.swift`:
- Type `$ command` to send bash commands
- Visual mode indicator: green accent, `$` prefix shown, terminal icon on send button
- Border color shifts green in bash mode
- Routes through `connection.runBash()` instead of `sendPrompt()`
- Empty state hint: "Tip: prefix with $ for shell commands"

### 5. Thinking Level Tracking (DONE)
- `ThinkingLevel.next` computed property for cycling
- `ServerConnection.thinkingLevel` tracks client-side state
- Updated from `rpc_result` data when server returns level
- Falls back to local cycling when server doesn't include data
- Resets to `.medium` on session switch

### 6. Token Formatting Extracted (DONE)
- `formatTokenCount()`, `inferContextWindow()`, `shortModelName()` extracted to internal functions in SessionToolbar.swift
- Removed duplicate private copies from ChatView.swift
- Shared by SessionToolbar, ChatView (SessionEndedFooter), ModelPickerSheet

### 7. JSONValue Helpers (DONE)
- Added `objectValue` and `arrayValue` convenience properties
