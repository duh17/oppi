## Context

Phone-native flows for creating new skills and improving existing ones.
These connect the Skills tab to the session/chat system.

Depends on: TODO-f471ff40 (server session setup), TODO-2cd10bc4 (skills tab),
  TODO-6717558a (skill detail)
Tracker: TODO-992ad1a6

## What to do

### A. Skill creation sheet

New `ios/PiRemote/Features/Skills/SkillCreationSheet.swift`:

Presented when user taps "+" on Skills tab.

- Multi-line text field: "Describe what you want your agent to be able to do..."
- Suggested name field (editable, auto-kebab-cased from description)
- Model picker (reuse existing model selection)
- "Start Building" button

On tap:
1. `POST /sessions { purpose: "skill-creation", name: <name>, model: <model> }`
2. Navigate to chat view for the new session
3. First prompt is the user's description

### B. Refinement flow

On SkillDetailView, wire the "Refine" button:

1. Present a sheet: "What would you like to improve?"
   - Text field for feedback
   - Model picker
   - "Start Refining" button

2. On tap:
   - `POST /sessions { purpose: "skill-refinement", skillRef: { name }, model }`
   - Navigate to chat view
   - First prompt is the user's feedback

### C. Session badge

In SessionListView, show a badge on skill-related sessions:
- "🔧 Building: data-viz" for creation sessions
- "🔧 Refining: data-viz" for refinement sessions

Comes from the session's `purpose` and `name` fields.
